HTML

  • Hyper Text Markup Language
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Usually holds the metadata elements
  • Handles content, strucutre, and meaning of each element

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline css is use of "style" attriute for specific HTML tag
  • Internal css style sheet embeds entire stylesheeet directly into HTML file using style element within head element
  • External css style links a file using link element within HTML element
  • utilize grouping with elements that share same styling using a comma
  • Declarations contain two components; CSS property to apply and the value of property

Git

  • Git is a open source version control systsem
  • allows to revert to previous code for debugging
  • git status: checks what branch we are currently working in
  • git checkout -b branch name: creates a new branch and switches to it
  • git command must precede every git statement

Js

  • A variable is a named container that allows us to store data in our code.
  • Conotrol flow is the order in which a computer executes code in a script.
  • Utilizes arrays a single variable used to group related data
  • 'console.log' method to output a message to the web console by adding an argument
